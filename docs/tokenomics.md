# ZED20 Token — Tokenomics

ZED20 Token is a BEP-20 utility token on Binance Smart Chain with a fixed, transparent supply model designed for DeFi interoperability and cross-chain utility.

## Token Specifications

- **Name**: ZED20 Token  
- **Symbol**: ZED20  
- **Blockchain**: Binance Smart Chain (BSC)  
- **Standard**: BEP-20  
- **Decimals**: 8  
- **Total Supply**: 200,000,000 ZED20  
- **Contract Address**: [`0x12c860695289e06b390e008ed0871172a24f5008`](https://bscscan.com/token/0x12c860695289e06b390e008ed0871172a24f5008)

## Supply Distribution

- **100% Initial Allocation**: All tokens were minted at deployment and initially held in the deployer wallet.
- **No team allocation, no pre-mine, no reserve wallet.**
- **No future minting or burning mechanisms.**

## Token Utility

- DeFi integration (liquidity provision, yield farming)
- Cross-chain bridging (planned for Phase 2)
- Settlement layer for dApps and institutional workflows
- Developer-friendly base asset for BSC applications

## Fees & Economics

- **Protocol Fees**: Zero  
- **Gas Fees**: Standard BSC transaction fees only  
- **Inflation**: None — fixed supply ensures long-term predictability
